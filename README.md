## Getting Started
#### Installation
``` sh
  # Clone this repo
  $ git clone https://github.com/devigor/wolf-boilerplate/ 
  $ cd wolf-boilerplate
  
  # Install Gulp Globally
  $ [sudo] npm install -g gulp
  
  # Install dependencies
  $ npm install
  
  # Commands 
  $ npm run dev // Development Mode
  $ npm run build // Production Mode
